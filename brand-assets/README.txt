USAGE NOTES
-----------
• SVG is the master. Regenerate any new raster sizes from the SVG so exports stay consistent.
• Use the LOCKUP where space allows; the SYMBOL for favicons/app icons/avatars; the
  WORDMARK only in single-colour or tightly constrained contexts.
• The lockup & wordmark SVGs use live Sora/Space Mono text (web-font embedded so they
  render in any browser). For print or logo databases, open in Illustrator/Inkscape and
  "Create Outlines" / "Object to Path" to make the type resolution-independent.
• Minimum sizes: symbol 16 px; lockup 24 px / 10 mm. Clear space = one bee-height all round.
• Never stretch, rotate, recolour, add effects, or rebuild the mark in another typeface.
